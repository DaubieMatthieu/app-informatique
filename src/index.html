InfiniteSense c'est quoi ?

InfiniteSense est un produit permettant de réaliser différents tests psychotechniques, comme des tests de reflexes, des tests d'écoute, etc... Il a été réalisé en partenariat avec Kinesense et Infinite Measures.

Le site internet où vous vous trouvez permet de naviguer dans une interface graphique qui a différents buts. Premièrement, le site permet d'avoir un compte protégé qui est utilisé pour réaliser les tests psychotechniques. Ainsi vous pouvez visualiser les résultats de vos tests à tout moment. Bien entendu, vous êtes le seul à pouvoir visualiser vos résulats sans aucune interférence.
Pour pouvoir réaliser un test, vous devez posséder un compte. Ce compte privé ne peut être obtenu que par une demande auprès d'un partenaire possèdant le matériel de test. Seulement dans ce cas vous pourrez confirmer votre inscription et vous connecter.

Un test peut être réalisé seulement par un partenaire qui possède le matériel adéquat. La technologie utilisé permet de transférer automatiquement les données des tests sur le site internet. Ainsi vous pourrez observer immédiatement les résultats de vos tests.
